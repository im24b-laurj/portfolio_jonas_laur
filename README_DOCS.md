## 🚀 Deployment Guides

### Zu Vercel deployen
```
1. Erstelle Account auf vercel.com
2. npm install -g vercel
3. vercel
4. Folge den Prompts
```

### Zu Netlify deployen
```
1. Erstelle Account auf netlify.com
2. Connecte dein GitHub
3. Select Portfolio Repository
4. Build startet automatisch
```

### Selbst hosten
```
1. npm run build
2. npm start
3. Auf deim Server hochladen
```
